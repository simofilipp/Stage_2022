Geopolitical Earth Planet is a 3D model of planet earth with 6 geo-political layers: Photo, Population, Science, Transport, Risk, Climate and all World Countries.

Each layer represents information corresponding to its name taken from open sources in Wikipedia, and also has a corresponding visualization shader. Also, the earth rotates in accordance with the time of year, taking into account the hours. You can select different layers and see specific information at the selected point on the earth. You can Select each Country and change all specific Data

Features:
- Mesh Based Countries
- 6 Unique Info Layers
- Screen space shader for Borders
- Well optimized shaders and data computing
- Intuitive, familiar interface
- Full source-code, well commented and named
- Custom data available for all layers and countries

Quick Guide:
Paste Prefab $Earth to your scene, edit WorldMapManager component if you wish to change any data. Paste Prefab Canvas to use UI layout for TimeManager and Layers change. Use CameraControllerInSpace Component on your mainCamera if you want.
